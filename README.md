# convolvr
WebVR game engine, server and virtual reality authoring tool.
#### Installation:
```
  git clone https://github.com/SpaceHexagon/convolvr.git
  cd convolvr/client
  npm install
  ./build.sh # or watchman src ./build.sh
  cd ../cmd
  go build
  ./cmd # starts server on default port 3007

```
